# EM::Mount

`EM::Mount` provides a simple abstraction to the `mount` system
command for eventmachine.

## Installation

Add this line to your application's Gemfile:

    gem 'em-mount'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install em-mount
